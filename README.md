# ฆ้องวงจำลองอัจฉริยะ

## วิธีเปิดใช้งาน
1. แตกไฟล์ ZIP
2. เปิดโฟลเดอร์ด้วยโปรแกรม Visual Studio Code
3. แนะนำให้เปิดผ่านส่วนเสริม Live Server หรืออัปโหลดขึ้น GitHub Pages / Netlify / Firebase Hosting
4. เมื่อเปิดผ่าน HTTPS บนโทรศัพท์ สามารถกด “ติดตั้งแอป” เพื่อใช้งานแบบ PWA และเปิดออฟไลน์ได้

## ฟังก์ชัน
- เล่นฆ้องวงใหญ่จำลอง 16 ลูก
- เล่นอิสระ
- ฝึกตามแนวโน้ต
- ฟังตัวอย่างอัตโนมัติ
- โหมดทดสอบ
- ปรับความเร็ว
- บันทึกคะแนนในเครื่อง
- รองรับโทรศัพท์ แท็บเล็ต และคอมพิวเตอร์
- ติดตั้งเป็น PWA และใช้งานออฟไลน์

## การเพิ่มเพลง
แก้ไขตัวแปร SONGS ในไฟล์ app.js

ตัวอย่าง:
{
  id: "song_1",
  name: "ชื่อเพลง",
  notes: [1,2,3,5,6,5,3,2]
}

หมายเลข 1–16 หมายถึงลูกฆ้องจากเสียงต่ำไปสูง

## หมายเหตุสำคัญ
แนวเพลงลาวเสี่ยงเทียนและระบำสุโขทัยในต้นแบบเป็น “แนวฝึกตัวอย่าง” ไม่ใช่โน้ตฉบับมาตรฐานของครูผู้สอน
เสียงในรุ่นนี้เป็นเสียงสังเคราะห์จาก Web Audio API เพื่อให้ใช้งานได้ทันที หากต้องการเสียงฆ้องจริง สามารถเปลี่ยนเป็นไฟล์เสียง .mp3 หรือ .wav ของลูกฆ้องแต่ละลูกได้ภายหลัง
